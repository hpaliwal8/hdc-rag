# Week-by-Week Plan

## Week 1 — Design + Data

1. Finalize dataset subsets (HotpotQA dev 400–600, TruthfulQA 100).
2. Define hallucination taxonomy and label schema.
3. Define prompt variants and baseline instructions.
4. Prepare loaders + sampling scripts.

## Week 2 — Model Runs

1. Run 4 models × 3 prompts on HotpotQA subset.
2. Run 4 models × 3 prompts on TruthfulQA subset.
3. Validate outputs + logging schema consistency.

## Week 3 — Labeling + Analysis

1. Label hallucination types (manual or assisted).
2. Compute hallucination rates and distributions.
3. Compare prompt variants across models.

## Week 4 — Write-up + Presentation

1. Write methodology + evaluation sections.
2. Generate tables/plots.
3. Error analysis and qualitative cases.
4. Final report + slides.
